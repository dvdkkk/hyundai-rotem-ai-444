가장 빠르게 문제를 해결할 방법들을 정리해 드립니다.

해결 방법 1: 브라우저 새로고침 (가장 추천)

현재 브라우저 화면을 새로고침(F5 버튼) 해주세요.
일시적으로 지연되는 부분이라, 페이지를 새로고침하면 파일 탐색기에 app_bundle.html 파일이 나타날 확률이 매우 높습니다. 나타나면 마우스 오른쪽 버튼으로 클릭하여 Download 하시면 됩니다.

해결 방법 2: 안 될 경우

만약 새로고침을 해도 보이지 않는다면, 기술적으로 **너무 큰 파일(650KB 이상의 실제 코드 Bundle)**이라 파일 시스템에서 인식은 되지만 UI 렌더링에서 빠질 수도 있습니다.

이 경우, 직접 파일을 다운로드할 수 있도록 작은 HTML 파일로 내용을 보실 수 있게 처리할 예정입니다.

  1. 브라우저 새로고침을 먼저 부탁드립니다.

  2. 그래도 보이지 않으시면 다시 말씀해 주세요. 다른 방식으로 파일을 가져가실 수 있게 도와드리겠습니다.
